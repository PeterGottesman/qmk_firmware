# [Peter Gottesman's](https://github.com/petergottesman) GH60 layout

Layout derived from the default GH60 keymap.

## Layers

### Base Layer
```
,-----------------------------------------------------------.
|Esc | 1|  2|  3|  4|  5|  6|  7|  8|  9|  0|  -|  =|Backsp |
|-----------------------------------------------------------|
|FN   |  Q|  W|  E|  R|  T|  Y|  U|  I|  O|  P|  [|  ]|  \  |
|-----------------------------------------------------------|
|Alt    |  A|  S|  D|  F|  G|  H|  J|  K|  L|  ;|  '|Return |
|-----------------------------------------------------------|
|Shift   |  Z|  X|  C|  V|  B|  N|  M|  ,|  .|  /|  Shift   |
|-----------------------------------------------------------|
|Ctrl|Gui |Alt |      Space           |CTRL |Gui |Alt | FN2 |
`-----------------------------------------------------------'
```

```
,-----------------------------------------------------------.
|GRV|F1 |F2 |F3 |F4 |F5 |F6 |F7 |F8 |F9 |F10|F11|F12|   DEL |

|    |   | UP|   |   |   |   |   |TAB|   |   |   |    |     |
|-----------------------------------------------------------|
|      |  L| DN| R |   |   |   |       |   |   |   |        |
|-----------------------------------------------------------|
|CAPS    |   |   |   |   |   |   |   |   |   |   |    |     |
|-----------------------------------------------------------|
|    |    |    |          Tilde         |    |    |    |    |
`-----------------------------------------------------------'
```
